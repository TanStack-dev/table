## テーブルステート (Vanilla JS) ガイド

TanStack Tableのコアは**フレームワーク非依存 (framework agnostic)** です。つまり、使用するフレームワークに関係なくAPIは同じです。各フレームワーク向けのアダプターが提供されており、テーブルコアを簡単に扱えるようになっています。利用可能なアダプターについては、Adaptersメニューを参照してください。
